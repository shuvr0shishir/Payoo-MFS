# Payoo Mobile Financial Services

Payoo is a simple web application inspired by mobile financial services like bKash. It allows users to perform basic financial operations through a clean and interactive interface.

## 🚀 Features

- Add money to account  
- Cash out functionality  
- Send money to another user  
- View transaction history  
- Simple and user-friendly interface  
- Responsive design for different devices
